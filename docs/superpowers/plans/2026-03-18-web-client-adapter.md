# Web Client Adapter & SvelteKit Scaffold — Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a ConnectRPC HTTP handler to the gateway process and scaffold a minimal SvelteKit app that proves the full browser-to-core pipeline works end-to-end.

**Architecture:** The gateway process gains an HTTP server hosting a ConnectRPC handler (implementing `WebService` proto) alongside a static file server for the SvelteKit build. The SvelteKit app uses generated TypeScript ConnectRPC clients to call the gateway. The handler translates browser requests into gRPC calls against the existing core service. A Playwright E2E test proves the full pipeline.

**Tech Stack:** Go 1.24, ConnectRPC (`connectrpc.com/connect`), protobuf, SvelteKit (`@sveltejs/kit` + `@sveltejs/adapter-static`), ConnectRPC TypeScript (`@connectrpc/connect`, `@connectrpc/connect-web`), pnpm, buf CLI, Playwright

**Spec:** `docs/specs/2026-03-18-web-client-adapter-design.md`

**Epic:** `holomush-qve`

## Chunk 0: Prerequisite — Introduce CharacterRef & Enrich Event Payloads

**Design Decision:** See `docs/specs/2026-03-18-web-client-adapter-design.md` §CharacterRef in Core Package.

The engine currently passes character data as loose primitives (`charID ulid.ULID`,
`charName string`, `locationID ulid.ULID`) — duplicating fields from `world.Character`.
A direct import of `world` into `core` would create a circular dependency. This chunk
introduces `core.CharacterRef` as a lightweight value type, refactors all engine method
signatures to accept it, enriches say/pose payloads with `character_name`, and updates
all callers (gRPC server, telnet handler).

package core

import "github.com/oklog/ulid/v2"

// CharacterRef is a lightweight reference to a character, used by engine
// methods to avoid passing loose primitives (charID, charName, locationID).
// This type lives in core to avoid circular imports with the world package.
// world.Character can convert to CharacterRef at package boundaries.
type CharacterRef struct {
	ID         ulid.ULID
	Name       string
	LocationID ulid.ULID
}

// String returns the character's display name.
func (c CharacterRef) String() string {
	return c.Name
}

Create `api/proto/holomush/web/v1/web.proto` with the `WebService` definition from the design spec. This is the browser-facing API — intentionally simpler than the internal core proto. No `RequestMeta`/`ResponseMeta`, no bytes payloads, flat message structures.

```protobuf
// SPDX-License-Identifier: Apache-2.0
// Copyright 2026 HoloMUSH Contributors

syntax = "proto3";

package holomush.web.v1;

option go_package = "github.com/holomush/holomush/pkg/proto/holomush/web/v1;webv1";

service WebService {
  // Authenticate as guest or registered user.
  rpc Login(LoginRequest) returns (LoginResponse);

  // Send a game command (say, pose, quit, etc.)
  rpc SendCommand(SendCommandRequest) returns (SendCommandResponse);

  // Server-streaming event feed. Client receives game events
  // (say, pose, arrive, leave) as they occur.
  rpc StreamEvents(StreamEventsRequest) returns (stream GameEvent);

  // Disconnect ends the session and triggers cleanup.
  rpc Disconnect(DisconnectRequest) returns (DisconnectResponse);
}

message LoginRequest {
  string username = 1;
  string password = 2;
}

message LoginResponse {
  bool success = 1;
  string session_id = 2;
  string character_name = 3;
  string error_message = 4;
}

message SendCommandRequest {
  string session_id = 1;
  string text = 2;
}

message SendCommandResponse {
  bool success = 1;
  string output = 2;
  string error_message = 3;
}

message StreamEventsRequest {
  string session_id = 1;
}

message GameEvent {
  string type = 1;
  string character_name = 2;
  string text = 3;
  int64 timestamp = 4;
}

message DisconnectRequest {
  string session_id = 1;
}

message DisconnectResponse {}
```

```go
// SPDX-License-Identifier: Apache-2.0
// Copyright 2026 HoloMUSH Contributors

// Package web implements the ConnectRPC web adapter for the HoloMUSH gateway.
package web

import (
	"context"
	"log/slog"
	"time"

	"connectrpc.com/connect"
	"github.com/samber/oops"

	corev1 "github.com/holomush/holomush/pkg/proto/holomush/core/v1"
	webv1 "github.com/holomush/holomush/pkg/proto/holomush/web/v1"
	"github.com/holomush/holomush/pkg/proto/holomush/web/v1/webv1connect"
)

const rpcTimeout = 10 * time.Second

// CoreClient is the gRPC interface used by Handler to communicate with
// the core service. Same methods as cmd/holomush/deps.go:GRPCClient.
type CoreClient interface {
	Authenticate(ctx context.Context, req *corev1.AuthRequest) (*corev1.AuthResponse, error)
	HandleCommand(ctx context.Context, req *corev1.CommandRequest) (*corev1.CommandResponse, error)
	Subscribe(ctx context.Context, req *corev1.SubscribeRequest) (corev1.Core_SubscribeClient, error)
	Disconnect(ctx context.Context, req *corev1.DisconnectRequest) (*corev1.DisconnectResponse, error)
}

// Handler implements the ConnectRPC WebService by delegating to the core gRPC client.
type Handler struct {
	client CoreClient
}

var _ webv1connect.WebServiceHandler = (*Handler)(nil)

// NewHandler creates a new ConnectRPC handler backed by the given core gRPC client.
func NewHandler(client CoreClient) *Handler {
	return &Handler{client: client}
}

// Login authenticates a user (guest or registered) via the core service.
func (h *Handler) Login(
	ctx context.Context,
	req *connect.Request[webv1.LoginRequest],
) (*connect.Response[webv1.LoginResponse], error) {
	rpcCtx, cancel := context.WithTimeout(ctx, rpcTimeout)
	defer cancel()

	resp, err := h.client.Authenticate(rpcCtx, &corev1.AuthRequest{
		Username: req.Msg.GetUsername(),
		Password: req.Msg.GetPassword(),
	})
	if err != nil {
		slog.Error("web: authenticate RPC failed", "error", err)

// StreamEvents opens a server-streaming event feed by subscribing to the
// core event stream and translating core events to GameEvent messages.
// Follows the same payload translation pattern as
// internal/telnet/gateway_handler.go:304-342.
func (h *Handler) StreamEvents(
	ctx context.Context,
	req *connect.Request[webv1.StreamEventsRequest],
	stream *connect.ServerStream[webv1.GameEvent],
) error {
	coreStream, err := h.client.Subscribe(ctx, &corev1.SubscribeRequest{
		SessionId: req.Msg.GetSessionId(),
	})
	if err != nil {
		return connect.NewError(connect.CodeInternal, oops.Errorf("subscribe failed"))
	}

	for {
		ev, recvErr := coreStream.Recv()
		if recvErr != nil {
			return nil // Stream ended (EOF or error) — close cleanly
		}

		gameEvent := translateEvent(ev)
		if gameEvent == nil {
			continue // Unknown event type — silently drop (forward compatibility)
		}

		if sendErr := stream.Send(gameEvent); sendErr != nil {
			return nil // Client disconnected
		}
	}
}

// Disconnect ends the session and triggers cleanup (leave events, guest release).
func (h *Handler) Disconnect(
	ctx context.Context,
	req *connect.Request[webv1.DisconnectRequest],
) (*connect.Response[webv1.DisconnectResponse], error) {
	rpcCtx, cancel := context.WithTimeout(ctx, rpcTimeout)
	defer cancel()

	if _, err := h.client.Disconnect(rpcCtx, &corev1.DisconnectRequest{
		SessionId: req.Msg.GetSessionId(),
	}); err != nil {
		slog.Error("web: disconnect RPC failed", "session_id", req.Msg.GetSessionId(), "error", err)
	}

	return connect.NewResponse(&webv1.DisconnectResponse{}), nil
}

// translateEvent converts a core Event to a web GameEvent.

func translateEvent(ev *corev1.Event) *webv1.GameEvent {

	if ev.GetTimestamp() != nil {

			return nil
		}

			Type:          "say",
			CharacterName: p.CharacterName,
			Text:          p.Message,
			Timestamp:     ts,
		}

	case "pose":
		var p posePayload
		if err := json.Unmarshal(ev.GetPayload(), &p); err != nil {
			slog.Error("web: failed to unmarshal pose payload", "error", err)
			return nil
		}
		return &webv1.GameEvent{
			Type:          "pose",
			CharacterName: p.CharacterName,
			Text:          p.Action,
			Timestamp:     ts,
		}

	case "arrive":
		var p arriveLeavePayload
		if err := json.Unmarshal(ev.GetPayload(), &p); err != nil {
			slog.Error("web: failed to unmarshal arrive payload", "error", err)
			return nil
		}

package web

import (
	"embed"
	"io/fs"
	"net/http"
	"os"
	"path/filepath"
	"strings"
)

//go:embed all:dist
var embeddedFS embed.FS

// FileServer returns an http.Handler that serves static files.
// If webDir is non-empty, files are served from that filesystem directory.
// Otherwise, files are served from the embedded dist/ directory.
// Unknown paths fall back to index.html for SPA client-side routing.
func FileServer(webDir string) http.Handler {
	var root fs.FS
	if webDir != "" {
		root = os.DirFS(webDir)
	} else {
		sub, _ := fs.Sub(embeddedFS, "dist")
		root = sub
	}

	fileServer := http.FileServer(http.FS(root))

	return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		// Clean the path
		path := strings.TrimPrefix(r.URL.Path, "/")
		if path == "" {
			path = "index.html"
		}

		// Try to open the file. If it exists, serve it.
		f, err := root.Open(path)
		if err == nil {
			f.Close()
			fileServer.ServeHTTP(w, r)
			return
		}

		// File not found — serve index.html for SPA fallback.
		// Only fallback for paths without file extensions (not missing assets).
		if filepath.Ext(path) == "" {
			r.URL.Path = "/"
			fileServer.ServeHTTP(w, r)
			return
		}

		http.NotFound(w, r)
	})
}

// SPDX-License-Identifier: Apache-2.0
// Copyright 2026 HoloMUSH Contributors

package web

import (
	"net/http"
	"slices"
	"strings"
)

// connectHeaders are the headers required by the Connect protocol.
var connectHeaders = []string{
	"Content-Type",
	"Connect-Protocol-Version",
	"Connect-Timeout-Ms",
	"Grpc-Timeout",
}

// CORSMiddleware returns an http.Handler that adds CORS headers for the
// given allowed origins. If origins is empty, no CORS headers are added.
// Handles preflight (OPTIONS) requests for ConnectRPC.
func CORSMiddleware(origins []string, next http.Handler) http.Handler {
	if len(origins) == 0 {
		return next
	}

	allowHeaders := strings.Join(connectHeaders, ", ")

	return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		origin := r.Header.Get("Origin")
		if origin == "" || !slices.Contains(origins, origin) {
			next.ServeHTTP(w, r)
			return
		}

		w.Header().Set("Access-Control-Allow-Origin", origin)
		w.Header().Set("Access-Control-Allow-Methods", "GET, POST, OPTIONS")
		w.Header().Set("Access-Control-Allow-Headers", allowHeaders)
		w.Header().Set("Access-Control-Max-Age", "3600")

		if r.Method == http.MethodOptions {
			w.WriteHeader(http.StatusNoContent)
			return
		}

		next.ServeHTTP(w, r)
	})
}

// SPDX-License-Identifier: Apache-2.0
// Copyright 2026 HoloMUSH Contributors

package web

import (
	"context"
	"errors"
	"log/slog"
	"net"
	"net/http"

	"github.com/holomush/holomush/pkg/proto/holomush/web/v1/webv1connect"
)

// Config holds configuration for the web HTTP server.
type Config struct {
	Addr        string
	Handler     *Handler
	WebDir      string
	CORSOrigins []string
}

// Server is the web HTTP server hosting ConnectRPC and static files.
type Server struct {
	httpServer *http.Server
	listener   net.Listener
	errCh      chan error
}

// NewServer creates a web server with ConnectRPC handler and static file server.
func NewServer(cfg Config) *Server {
	mux := http.NewServeMux()

	// Register ConnectRPC handler
	path, connectHandler := webv1connect.NewWebServiceHandler(cfg.Handler)
	mux.Handle(path, connectHandler)

	// Register static file server as fallback
	mux.Handle("/", FileServer(cfg.WebDir))

	// Wrap with CORS if origins configured
	var handler http.Handler = mux
	if len(cfg.CORSOrigins) > 0 {
		handler = CORSMiddleware(cfg.CORSOrigins, mux)
	}

	return &Server{
		httpServer: &http.Server{
			Addr:    cfg.Addr,
			Handler: handler,
		},
		errCh: make(chan error, 1),
	}
}

// Start begins listening and serving. Returns an error channel for
// background monitoring (same pattern as observability.Server).
func (s *Server) Start() (<-chan error, error) {
	ln, err := net.Listen("tcp", s.httpServer.Addr)
	if err != nil {
		return nil, err
	}
	s.listener = ln

	go func() {
		defer close(s.errCh)
		if err := s.httpServer.Serve(ln); err != nil && !errors.Is(err, http.ErrServerClosed) {
			slog.Error("web: HTTP server error", "error", err)
			s.errCh <- err
		}
	}()

	slog.Info("web HTTP server started", "addr", s.Addr())
	return s.errCh, nil
}

// Stop gracefully shuts down the HTTP server.
func (s *Server) Stop(ctx context.Context) error {
	return s.httpServer.Shutdown(ctx)
}

// Addr returns the actual address the server is listening on.
// Useful when started with ":0" for random port selection.
func (s *Server) Addr() string {
	if s.listener != nil {
		return s.listener.Addr().String()
	}
	return s.httpServer.Addr
}
```

  import { createClient } from "@connectrpc/connect";
  import { transport } from "$lib/transport";
  import { WebService } from "$lib/connect/holomush/web/v1/web_pb";

  const client = createClient(WebService, transport);

  let characterName = "";

      if (resp.success) {
        sessionId = resp.sessionId;
        characterName = resp.characterName;
        connected = true;
        startEventStream();

      error = `Connection error: ${e}`;
    }

    try {
      for await (const event of client.streamEvents({ sessionId })) {
        events = [
          ...events,

            characterName: event.characterName,
            text: event.text,
          },
        ];
      }
    } catch {
      // Stream ended
    }
  }

      commandText = "";
    } catch (e) {
      error = `Command error: ${e}`;
    }
  }

  async function disconnect() {
    try {
      await client.disconnect({ sessionId });
    } catch {
      // Best effort
    }
    connected = false;
    sessionId = "";
    events = [];
  }
