<threat_model>

### Trust Boundaries

| Boundary | Description |
|----------|-------------|
| emit path → EventBus | A plugin could forge actor identity or emit sensitive content without declaration |
| history read → plugin boundary | A non-member could read private channel history |
| audit redelivery → channel_log | Duplicate deliveries could double-write history |

### STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-01-01 | Information disclosure | non-member reads private channel history | high | mitigate | QueryHistory membership auth step-1 BEFORE any DB work; caller kind/id validated; PermissionDenied on non-member |
| T-01-04 | Spoofing | forged actor kind on emit | medium | mitigate | actor_kinds_claimable: [plugin, character] gate at event_emitter.go::Emit (both runtimes); emit fence self-gates |
| T-01-17 | Tampering | duplicate audit rows on redelivery | low | mitigate | Idempotent INSERT ... ON CONFLICT (id) DO NOTHING (id is the bus dedup key) |
| T-01-18 | Information disclosure | wildcard/malformed subject probing | medium | mitigate | parseChannelSubject rejects wildcards (InvalidArgument); generic error messages, no inner-detail leak |
| T-01-05 | Denial of service | unbounded history page reads | medium | mitigate | Page size clamped to scrollback_cap (500); joined_at floor bounds the readable window (D-07) |
| T-01-SC | Tampering | package installs | n/a | accept | No package installs; no crypto.emits so crypto-reviewer does not fire (D-04); SC checkpoint does not fire |

</threat_model>
