<objective>
Delete `core.Event`, `core.NewEvent`, and `core.EventAppender` (D-01), collapse
the duplicate actor bridges (D-06), and amend the repo rules that mandate the
deleted symbols.

Purpose: this is the requirement. ARCH-04 = "a single Event representation
exists; the `core.Event`/`eventbus.Event` duplication is gone, all callers
migrated, with no behavior change." Phase 5 already fixed the direction —
`internal/world/outbox/wire.go:154` converts `wmodel.Envelope` →
`eventbus.Event`; `eventbus.Event` is the survivor.

Output: one Event type, one actor bridge, and rules that describe the code.
</objective>

<constraints>
### `core.Event` cannot move to a neutral leaf — the type stays in `internal/eventbus`

`eventbus.Event` keeps an **unexported `auditRow`** field with a
package-internal accessor (`AuditRowOf`, `audit_row_access.go`) consumed by
`history.PluginDowngradeFence` for **INV-CRYPTO-42/50**. Moving the type out of
`internal/eventbus` breaks that package-private seam and silently breaks a
crypto downgrade fence. This is why D-01 **rejected** a neutral leaf package for
the whole Event struct. Only the **vocabulary** became a leaf (07-02).

### `core.Actor` is NOT deleted — D-06's "collapse with core.Actor" means the BRIDGES collapse

D-01's deletion list names exactly three symbols: `core.Event`, `core.NewEvent`,
`core.EventAppender`. `core.Actor` is **not** on it, and deleting it is
**provably impossible** within D-01's own constraints. Verified live:
`internal/command/dispatcher.go:427` calls
`core.WithActor(ctx, core.Actor{Kind: core.ActorCharacter, ...})` — `core.Actor`
is the actor-**context** vocabulary carried on `context.Context` through dispatch
into plugin emit. `internal/command` MUST NOT import `internal/eventbus` (D-01),
so `core.Actor` cannot become `eventbus.Actor`.

**SETTLED:** `core.Actor`, `core.ActorKind`, `core.ActorSystemID`,
`core.WithActor`, `core.ActorFromContext` stay in `internal/core`. D-06 is
honored by collapsing the **three duplicate bridges to one** — which is exactly
what D-06 enumerates (`coreToBusActor`, `coreActorToEventbusActor`,
`harnessCoreToBusActor`), two of which are self-admitted hand-copies.

### Scope fence

`internal/plugin/hostcap/servers.go` is `internal/grpc/focus`-coupled
(MEDIUM-4). Touching it may surface Phase 8 territory — change only the Event
type and the `ReplayTail` shape. Do not unwind the coupling.
</constraints>
