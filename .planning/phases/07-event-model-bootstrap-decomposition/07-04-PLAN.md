<objective>
Make ARCH-05 permanent: forbid `internal/core`, `internal/session`, and
`internal/grpc` wholesale (D-15/D-17), add the **transitive-closure** gate the
existing AST test structurally cannot express (FINDING-3), and bind
`INV-EVENTBUS-1` genuinely.

Purpose: `cmd/holomush/gateway_imports_test.go`'s `checkFile` iterates
`file.Imports` — **direct imports only** (`:148`, verified live). D-15's rule
("MUST NOT import any package that *transitively* reaches DB/domain/bus") is a
closure property. Without the closure gate, a future gateway file could import
an innocuous-looking leaf that later grows a `world` dep and the gate stays
green. The closure test is also what makes `INV-EVENTBUS-1`'s binding **genuine
rather than partial**.

Output: a closure gate with a positive control, an amended `forbidden` list, and
`INV-EVENTBUS-1` flipped `pending` → `bound`.
</objective>
