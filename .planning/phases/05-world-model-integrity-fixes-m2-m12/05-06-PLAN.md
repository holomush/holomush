<objective>
Introduce the compile-time write-requires-envelope seam (`mutate(ctx, entity, expectedVersion, envelope)`), wire `MoveCharacter` end-to-end through the outbox, and delete the post-commit emit path outright — folding WR-01 (D-03). Output: the mutate() seam, MoveCharacter through the outbox, events.go/event_store_adapter.go deleted, chaos helpers + m2 spec rewritten, f1 doc corrected.
</objective>
