<objective>
Introduce the compile-time write-requires-envelope seam (`mutate(ctx, entity, expectedVersion, envelope)`), wire `MoveCharacter` end-to-end through the outbox, and delete the post-commit emit path outright — folding WR-01 (D-03). This is the pivotal slice-2 plan: the emit-path deletion, the first outbox mover, and the resilience-test rewrite MUST land together (Pitfall 3 — deleting `events.go` breaks the integration harness that `task test` does not compile).

Purpose: The one-pager makes the guarded write seam require the envelope so that "a stated principle silently unrealized" (the drift that caused the ADR) is structurally non-recurrable. `MoveCharacter` is the empirically characterized M2 window — moving it into the same-tx outbox closes that window and proves the mechanism on a real mover before the mechanical rollout.
Output: the mutate() seam, MoveCharacter through the outbox, events.go/event_store_adapter.go deleted, chaos helpers + m2 spec rewritten, f1 doc corrected.
</objective>
